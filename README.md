# 日程調整 Pro 📅

Google スプレッドシートをデータベースとして活用した、サーバーレスの日程調整アプリです。

## 🚀 公開URL
[ここにGitHub PagesのURLを貼る]

## ✨ 主な機能
- **3つのモード**: 時間帯調整（15分刻み）、時間割調整、複数候補アンケート。
- **スマート入力**: ドラッグで一気塗り、パレット移動、自動時間割反映（時間割パワー）。
- **完全レスポンス**: PCでもスマホでも快適に動作する固定ヘッダー・大型ボタン。
- **自動締め切り**: 設定した期限を過ぎると自動で回答不可に切り替え。

## 🛠 セットアップ (開発者向け)
このアプリは `stlite` を使用しているため、サーバーなしで動作します。

### ファイル構成
- `app.py`: メインのロジック（Python）
- `index.html`: GitHub Pagesで動かすためのStliteエントリーポイント
- Google Apps Script: スプレッドシートとの通信用（コードはリポジトリ外で管理）

### データベース
Google スプレッドシートの以下のシートを使用しています：
- `users`: ユーザー情報・権限
- `events`: イベント詳細・期限
- `responses`: 回答データ
- `fixed_schedule`: 各ユーザーの固定時間割
